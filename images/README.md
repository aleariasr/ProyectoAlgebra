# Carpeta de Imágenes de Entrada

Coloca aquí 3 imágenes a color (PNG/JPG) que usarás en el proyecto, por ejemplo:
- img1.png
- img2.jpg
- img3.jpg

## Notas

- Las imágenes deben ser archivos de imagen válidos (PNG, JPG, JPEG, BMP, GIF)
- Se recomienda usar imágenes a color para aprovechar todas las transformaciones
- Las imágenes procesadas se guardarán en la carpeta `outputs/`
- No subir imágenes con copyright no-permitido
